This folder contains everything you need to fill out the Google Play Store listing.

1.  **Text Fields:**
    -   `01_title_and_short_desc.txt`: Copy the Title and Short Description.
    -   `02_full_description.txt`: Copy the entire text into the "Full Description" box.
    -   `03_privacy_policy_content.txt`: Host this text online (e.g., Google Doc, GitHub Gist, or Privacypolicygenerator.info) and paste the LINK in the Console.

2.  **Images:**
    -   `icon_512.png`: Upload this as the App Icon (Hi-res).
    -   *Screenshots:* You must take these yourself using the emulator or your phone. Take at least 4:
        -   Home Screen (showing Prayer Cards).
        -   Qibla Compass.
        -   Settings (showing the options).
        -   Names of Allah grid.
    -   *Feature Graphic:* You need a 1024x500 image. You can create a simple blue/gold background with the text "Qiyam Dawud" using tools like Canva.

3.  **Testing:**
    -   Follow `04_testing_instructions.txt` to pass the 20-tester requirement.

Good luck!
